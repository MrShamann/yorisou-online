## 1. Short Invitation

**Context:** First outreach from Edward to a hand-selected candidate. Sent in a private conversation channel.

```
Yorisouで「詳細レポート」というものを準備しています。
今回、その読みやすさと安全性を確認するための、少人数のテスト読者を募集しています。

支払いは発生しません。
ご参加は任意で、お断りいただいてもかまいません。
詳しい内容をお伝えできますので、ご興味があればお知らせください。
```

---

## 2. Full Invitation

**Context:** Sent after the candidate replies with interest.

```
詳細レポートは、Yorisouのチェック結果をもとに、
いまの状態を「感情」「人との距離」「行動」「回復」の視点から整理する自己理解レポートです。

今回のテストは、

- 今回のYorisouチェックの回答をもとに、レポートを作ります
- 作成したレポートは、担当者(Edward)がひとつずつ読み確認してからお渡しします
- お渡し後、読みやすさや安全性について、いくつかご感想をお伺いします

このレポートは、医療・心理診断、治療、カウンセリング、未来予測ではありません。
将来の出来事を当てるものでもありません。
今の状態を整理し、小さな次の行動を考えるための読み物です。

支払いは発生しません。
将来、有料でご提供する場合の価格や条件は、現時点では決まっていません。
今回のご協力を、将来の購入の前提として扱うこともありません。
ご協力の有無は、今後のお付き合いに影響しません。
```

## 3. Eligibility Note

**Context:** Shown together with the full invitation. Plain language; no invasive questions.

```
ご参加にあたって、以下にあてはまる方をお招きしています。

- 18歳以上の方
- 日本語で日常的な読み書きができる方
- 今回のYorisouチェックに、ご自身で回答できる方
- 自由意志でご参加・ご辞退をご判断いただける方
- 報酬や引き換えを前提とせず、お読みいただける方
```

---

## 4. Not Eligible Note

**Context:** Honest, non-judgmental. The candidate is not asked invasive questions; Edward applies the exclusion judgment using existing knowledge.

```
今回のテストは、以下の状況にある方には合わない可能性があります。
別の道のほうがふさわしい場合があるため、無理にご参加いただかなくて大丈夫です。

- いま、強いつらさや危機的な状況にある方
- 安全でない、または相手のふるまいが怖いと感じる関係にある方
- いま治療や専門的な支援を受けていて、ご自身の支援に影響が出る可能性があると感じる方

ご自身で迷われる場合は、参加を見合わせていただいてかまいません。
```

## 5. Consent Preview

**Context:** Shown before the formal consent step. Outlines the two confirmations that will follow.

```
このあと、二つの確認をお願いします。

1. ご利用範囲の確認(どの情報をレポート作成に使うか)
2. レポートの位置づけの確認(このレポートが何で、何ではないか)

それぞれ、内容をお読みいただき、ご同意いただける場合のみ進みます。
ご同意いただかない選択もできます。その場合は、レポート作成には進みません。
```

## 6. No Payment / No Emergency / No Sensitive Details Note

**Context:** Shown alongside the consent preview. Three honest, plain statements that prevent miscalibration.

```
お伝えしておきたい点が三つあります。

支払いは発生しません。
このレポートは、緊急時の支援や相談窓口の代わりにはなりません。
今後のご記入の際、自由記述欄には、お名前、相手の方の特定につながる情報、
健康・危機の詳細、LINEやメッセージのやり取り内容は、書かないようお願いします。

これらの情報は、内部の改善でも扱いきれないため、削除させていただくことがあります。
```

## 7. Reply Template — Accepting Interest

**Context:** Reply Edward sends when a candidate confirms they would like to proceed after reading the full invitation.

```
ご検討ありがとうございます。
それでは、次のステップに進ませていただきます。

- まず、上記の「ご利用範囲の確認」と「レポートの位置づけの確認」をお願いします
- そのあと、Yorisouのチェックを受けていただきます
- チェック結果をもとに、こちらでレポートを作成し、確認後にお届けします

お渡しまでに、確認のお時間をいただきます。
当日中のお渡しは難しい場合があります。
途中で、保留や、お渡しを見合わせる判断をすることもあります。
その場合は、ご連絡します。
```

## 8. Reply Template — Declining or Asking to Stop

**Context:** Reply Edward sends when a candidate declines or asks to withdraw at any point.

```
お知らせいただきありがとうございます。
ご辞退を承りました。今回はご無理なさらず、どうぞご自身のペースを大切にしてください。

これまでにお預かりしていた情報の取り扱いについてご希望があれば、お知らせください。
利用を止めることや、削除のご依頼を承ります。
削除できる範囲や、確認に必要なお時間については、ご依頼後にご案内します。

今回ご検討いただいたこと、あらためてお礼申し上げます。
```

## 9. Level 1 Data-Use Confirmation

**Context:** First of the two formal consent blocks. Short version shown above checkbox, full version expandable.

### Short

```
今回のYorisouチェックの回答と結果を、詳細レポート作成に使います。
これは、今回の詳細レポートを作るためだけの利用です。
```

### Full (expand "詳しく見る")

```
このテストでは、次のものを使います。

- 今回のYorisouチェックの回答
- 今回のチェックの結果タイプと傾向
- 回答パターンから見える、いまの状態の特徴

次のものは使いません。

- 過去のYorisouチェック結果
- LINEでのやりとりや、メッセージの内容
- 他のサービスから集めた情報
- お名前、ご連絡先、その他の個人情報
- このレポートの内容を、AIモデルの学習に使うこと

詳細レポートは、担当者の確認を経てお届けします。
お届けまでに少しお時間をいただく場合があります。

同意は、いつでも取り下げることができます。
取り下げた場合、その時点以降の利用を止めます。
```

### Checkbox label

```
今回のチェック結果を、詳細レポート作成に使うことに同意します。
```

### Helper text (below checkbox)

```
これは購入や申し込みではありません。
このテストでは、支払いは発生しません。
```

### Error text (if user attempts to proceed unchecked)

```
レポート作成の前に、ご利用範囲のご確認とご同意をお願いします。
ご同意いただかない選択もできます。その場合は、レポート作成には進みません。
```

## 10. Report Boundary Confirmation

**Context:** Second of the two formal consent blocks.

### Short

```
詳細レポートは、いまの状態を整理するための自己理解レポートです。
医療・心理診断、治療、カウンセリング、未来予測ではありません。
```

### Full (expand "詳しく見る")

```
このレポートが行うこと:
- いまの状態の傾向を、いくつかの視点から整理する
- 小さな次の行動を考えるための入口を示す
- 自分の状態を見返すための要約を残す

このレポートが行わないこと:
- 医療や心理の診断
- 治療やカウンセリング
- 将来の予測や運命的な判断
- ご本人やまわりの方への、具体的な決定の指示
- 関係を続ける・終わらせる・連絡する・しないなどの判断
```

### Checkbox label

```
このレポートが診断・治療・未来予測ではなく、いまの状態を整理する読み物であることを理解しました。
```

### Helper text

```
読みながら気になる点や、合わないと感じる部分があれば、無理に当てはめなくて大丈夫です。
```

### Error text

```
お進みいただく前に、レポートの位置づけのご確認とご同意をお願いします。
ご同意いただかない選択もできます。その場合は、レポート作成には進みません。
```

## 11. Storage Option A Explanation

**Context:** First cohort uses Option A only. B and C are not displayed.

```
今回のテストでは、次の保管の取り扱いとなります。

- レポートを作成し、お渡ししたあとは、本文を保存しません。
- お渡しの際にお読みいただけますが、その後ご自身で控えを残されない場合、
  あとから同じレポートを再表示することはできません。
- 管理上必要な記録(レポートのIDや種類、安全確認の状況など)は残ります。
- ご本人のお名前や、レポート本文の内容は残しません。

ご希望があれば、ご自身の手元で控え(スクリーンショットや書き写し)を残していただいてかまいません。
```

### Confirmation label

```
保管の取り扱い(本文をお渡し後に保存しないこと)を理解しました。
```

**Context:** Last step before the candidate's request is sent into the workflow.

## 14. Wording Rules Summary

| Rule | Required posture |
|---|---|
| **No payment** | "支払いは発生しません" stated at invitation, full invitation, helper text, no-payment note |
| **No emergency support** | "緊急時の支援や相談窓口の代わりにはなりません" stated in the no-payment/no-emergency block |
| **No sensitive free-text details** | Free-text warning lists the categories not to include; appears at the no-sensitive-details note and again in feedback copy |
| **No LINE history** | "LINEでのやりとりやメッセージ内容は使いません" stated explicitly |
| **No memory** | "長期メモリーや過去のチェック結果は使いません" stated explicitly |
| **No raw private text** | "自由記述や個人的な詳細を、レポート作成に取り込むことはしません" stated explicitly |
| **No model training** | "AIモデルの学習に使うことはしません" stated explicitly |
| **Manual review delay** | "お渡しまでに、確認のお時間をいただきます" / "当日中のお渡しは難しい場合があります" |
| **User can stop / request non-use** | Stated in reply templates and in the decline acknowledgment |
| **No overpromise deletion** | "削除できる範囲や、確認に必要なお時間については、ご依頼後にご案内します" — never "すぐに削除" or "完全に削除" |
