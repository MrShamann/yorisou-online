# YORISOU Emotional Value Governance v1.0 (Core System 1)

**Status:** Approved
**Pack:** YORISOU Governance Pack v0.4.0 · **Drafted:** 2026-07-13 · **Approved as activation candidate:** 2026-07-14 · **Approver:** Edward

## 1. Purpose
Govern how YORISOU creates and expresses emotional value without manipulation.

## 2. Scope
All value-expressing copy, reflection prompts, and meaning-related features.

## 3. Binding rules
1. Emotional value arises from the user's own kept meanings, revisited with permission — never from generated flattery, scoring, or dependency loops.
2. No emotional scoring or ranking of users; no 'emotional health' metrics presented to users.
3. Reflection prompts follow the validated question form (今の気持ちにも近いですか-style: comparative, low-pressure, answerable with 'no').
4. 'Letting go' (もう手放してよい) is always an equally honored outcome.

## 4. Prohibited behavior
Flattery engines; emotional scoring; manufactured urgency; framing absence as loss.

## 5. Decision authority
Edward for principle changes; change-managed copy edits with language review.

## 6. Required evidence
Copy-governance lint in CI; reflection-choice distribution visible on the Founder Dashboard (aggregate).

## 7. Exceptions
None.

## 8. Change control
