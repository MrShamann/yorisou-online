## 1. Purpose

YORISOU is founder-led. This register records decisions that change strategy, governance, product scope, data use, commercial model, public representation, or execution authority.

## 2. Required Fields

| Field | Required content |
|---|---|
| Decision ID | Stable identifier |
| Date | YYYY-MM-DD |
| Decision owner | Edward unless delegated |
| Subject | Concise title |
| Context | Why a decision was needed |
| Options considered | Material alternatives |
| Decision | Exact approved direction |
| Scope | Files, product areas, or releases affected |
| Effective status | Draft / Approved / Reversed / Superseded |
| Conditions | Preconditions or limits |
| Follow-up | Required implementation or review |
| Evidence | Related files, PRs, tests, or reports |

## 3. Decisions Requiring Registration

Register decisions involving:

- market or brand change;
- test-methodology change;
- new user-data use;
- open social interaction capability;
- payment or commercial ranking;
- external data sharing;
- AI authority expansion;
- production release of high-risk capability;
- change of execution agent or approval authority.
